## 🚀 Overview

This project is a **console-based Library Management System** designed to manage books efficiently in a library environment.

It allows users to:
- Add new books 📚
- Delete books ❌
- Search for books 🔍
- Display all books 📖
- Store book location inside library 📍

---

## ✨ Features

✔ Add Book  
✔ Delete Book  
✔ View All Books  
✔ Search by ID / Name  
✔ Store Book Location  
✔ Simple and fast CLI interface
